
# TeaOpenPTY
TeaOpenPTY is a library for PHP to spawn an interactive Linux terminal.

# Installation
```
git clone https://github.com/ammarfaizi2/TeaOpenPTY
cd TeaOpenPTY
phpize
./configure
make
```

The shared library file will be located at `modules/tea_openpty.so`
